# Learn Kanji - JLPT N5 to N1 — App Landing Page

GitHub Pages site for the **Learn Kanji - JLPT N5 to N1** iOS app.

## Setup

1. Create a GitHub repo named `LearnKanjiApp` under `HarmonyAstroApps`
2. Push this folder to the `main` branch
3. Enable GitHub Pages → Source: `main` / `/ (root)`
4. Site will be live at: `https://harmonyastroapps.github.io/LearnKanjiApp/`

## Before publishing

- [ ] Drop real app screenshots into `public/images/screenshots/` (1.png – 5.png)
- [ ] Add a hero screenshot at `public/images/hero_screenshot.png`
- [ ] Add an OG image at `public/images/og_image.png` (1200×630)
- [ ] Update the App Store link (replace `id6744498124` with the real app ID)
- [ ] Update the canonical URL in `index.html` once the repo is live
